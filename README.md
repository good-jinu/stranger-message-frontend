# stranger-message-frontend

낯선 사람에게 무작위로 일방적인 편지를 보내는 서비스입니다.
물병편지에서 영감을 받았습니다. [제작자 블로그](https://blog.naver.com/kimgooni/120091467315)

## Getting Started

개발할 때 사용한 node js 버전은 다음과 같습니다.

- node == 18.13.0

패키지를 설치합니다.

```bash
npm install
# or
yarn install
```

개발서버를 실행합니다.

```bash
npm run dev
# or
yarn dev
```

어플리케이션을 빌드합니다.
```bash
npm run build
# or
yarn build
```

서버를 실행합니다.
```bash
npm run start
# or
yarn start
```
